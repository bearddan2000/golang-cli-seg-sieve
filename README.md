# golang-cli-seg-sieve

## Description
Given a range low, high,
print all primes in this range.

## Tech stack
- bash
- golang 1.13

## Docker stack
- docker-cli
- ubuntu:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
